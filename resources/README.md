# Resources

This folder contains reusable student resources, templates, references, and supporting materials for the IBS CU Boulder **Quantitative Data Management and Workflow for Social Scientists** microcredential.

## Student Setup And Troubleshooting

- `how-to-use-this-repository.md`: overview of the repository structure and how to move through modules.
- `student-stata-setup.md`: beginner-friendly guide to Stata, do-files, logs, and Stata setup.
- `student-r-setup.md`: beginner-friendly guide to R, RStudio/Positron, packages, scripts, and errors.
- `file-paths-and-folders.md`: guide to repository root, data folders, relative paths, and common Mac/Windows path issues.
- `troubleshooting.md`: common Stata and R problems with first fixes.

## Crosswalks And Templates

- `stata-r-crosswalk.md`: parallel Stata and R/tidyverse reference for common data wrangling tasks.
- `module-template.md`: standard module structure template.
- `repo-quality-checklist.md`: instructor-facing/manual repository QA checklist for launch and review.
