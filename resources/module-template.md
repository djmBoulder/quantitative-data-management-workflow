# Standard Module Template

Use this template when drafting full module materials. Keep Stata and R examples substantively parallel, use synthetic or publicly distributable data only, and explain why each workflow matters before showing code.

## 1. Why This Skill Matters

Briefly explain why the skill is important for reproducible social science research and where students will use it in later methods courses or research projects.

## 2. Conceptual Overview

Introduce the main idea in beginner-friendly language before moving into commands or functions. Define any terms students need in order to understand the workflow.

## 3. Key Stata Commands

List the Stata commands students will use in the module, with a short note about what each command does.

## 4. Key R Functions/Packages

List the R functions or packages students will use in the module, with a short note about what each function or package does.

## 5. Worked Example

Provide a guided Stata example and a substantively parallel R example using the same synthetic teaching dataset whenever possible.

## 6. Practice Task

Describe a short task students can complete on their own to apply the module skill.

## 7. Common Errors

List common beginner mistakes, how students can recognize them, and what they should check before moving on.

## 8. Competency Checklist

Provide a checklist students can use to confirm that they can perform the skill and explain the workflow in plain language.
