# 00 Orientation Checklist

Use this checklist before moving to Module 01.

## Repository Navigation

- [ ] I can find the repository root.
- [ ] I can find the `modules/` folder.
- [ ] I can find the `data/synthetic/` folder.
- [ ] I can find the `data/working/` folder.
- [ ] I can find the `data/output/` folder.
- [ ] I can find the `logs/` folder.
- [ ] I can find the course `README.md`.

## Module Structure

- [ ] I understand that each module has a `README.md`.
- [ ] I understand that each module has a `lesson.md`.
- [ ] I understand that Stata labs use `.do` files.
- [ ] I understand that R labs use `.R` files.
- [ ] I understand that `assignment.md` explains student work.
- [ ] I understand that `checklist.md` helps me self-check.
- [ ] I understand that `solution_notes.md` is instructor-facing.

## Data Safety

- [ ] I can identify the `synthetic_gss_like` teaching data.
- [ ] I understand that synthetic data are not real student or respondent data.
- [ ] I understand that raw or synthetic source files should not be manually edited.
- [ ] I understand that working files are generated by scripts.
- [ ] I understand that output files are generated reports, tables, and handoff files.
- [ ] I understand that generated working, output, and log files are normally ignored by Git.

## Software Pathway

- [ ] I know whether my section expects Stata, R, or dual Stata/R submissions.
- [ ] I understand that one primary pathway is sufficient unless my instructor requires both.
- [ ] I ran the Stata setup check, the R setup check, or both.
- [ ] My setup check found `data/synthetic/synthetic_gss_like_person_2024.csv`.
- [ ] My setup check wrote a report to `data/output/`.
- [ ] My setup check wrote a log to `logs/`.
- [ ] I know where to ask for help if my setup check did not run.

## Capstone Orientation

- [ ] I can explain that the final capstone is a data management portfolio.
- [ ] I understand that later modules build pieces of the capstone.
- [ ] I understand that scripts, documentation, QA checks, and workflow memos all matter.
- [ ] I can name one workflow habit I want to improve during the course.
