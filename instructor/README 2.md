# Instructor Materials

This folder contains instructor-facing guidance, teaching notes, QA checklists, and implementation materials for the IBS CU Boulder **Quantitative Data Management and Workflow for Social Scientists** microcredential.

Do not copy instructor-only guidance, grading notes, expected outputs, or solution details into student-facing files.

## Files

- `teaching-guide.md`: delivery guidance for noncredit microcredential, 1-credit, bootcamp, Stata-first, R-first, and dual-pathway formats.
- `module-prep-checklist.md`: module readiness checklist for instructors.
- `common-student-errors.md`: troubleshooting guide for common beginner issues.
- `software-setup-guide.md`: Stata and R setup assumptions, packages, paths, logs, and folders.
- `course-implementation-notes.md`: launch-readiness and course operations notes.
- `stata-lab-smoke-test-checklist.md`: manual pre-launch Stata lab smoke-test checklist.
- `r-lab-smoke-test-checklist.md`: manual pre-launch R lab smoke-test checklist.
